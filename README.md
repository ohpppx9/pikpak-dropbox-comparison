## Dropbox는 뭘 잘하나

Dropbox는 2008년부터 시작한 파일 동기화 서비스다. 핵심 개념은 "로컬 폴더와 클라우드를 실시간으로 같은 상태로 유지한다"는 것이다. 내 컴퓨터에서 파일을 수정하면 자동으로 클라우드에 올라가고, 다른 기기에서도 최신 상태를 볼 수 있다.

문서 협업, 팀 공유, 사무용 파일 관리 같은 용도에서 Dropbox는 여전히 강하다. PDF 편집, 전자 서명, 파일 링크 공유 같은 기능들이 잘 갖춰져 있고, 기업용 보안 설정도 촘촘하다.

다만 가격 대비 용량으로 따지면 솔직히 넉넉하지 않다. 무료 플랜은 **2GB**이고, 유료 Plus 요금제는 월 $9.99(연간 결제 기준)에 **2TB**를 제공한다.

## PikPak은 뭘 잘하나

PikPak은 2021년에 싱가포르 법인 PIKCLOUD PTE. LTD.가 출시한 클라우드 서비스다. Dropbox처럼 "내 파일을 올려서 보관한다"는 개념에서 출발하지만, 동작 방식이 근본적으로 다르다.

PikPak의 핵심 기능은 **클라우드 다운로드**다. 토렌트 마그넷 링크, ED2K 링크, HTTP URL, 트위터/틱톡 같은 SNS 영상 링크를 입력하면, PikPak 서버가 내 기기 대신 그 파일을 받아서 내 클라우드에 저장해준다. 내 인터넷 속도가 느려도, 내 기기가 꺼져 있어도 관계없다. 파일은 PikPak 서버에서 이미 저장을 완료한다.

저장된 영상은 유튜브처럼 스트리밍으로 볼 수 있고, 4K 원본 화질 재생도 지원된다. Android, iOS, Windows, macOS, 웹 브라우저, 심지어 Android TV와 Telegram 봇까지 접근 방법도 다양하다.

프리미엄 회원은 **10TB** 저장 공간을 쓸 수 있다. 연간 결제 기준으로 월 약 $4 수준이다. Dropbox Plus의 2TB와 비교하면 용량이 5배인데 가격은 비슷하거나 오히려 저렴한 수준이다.

## 결정적인 차이: 어떤 상황에서 어떤 걸 써야 할까

### PikPak이 맞는 사람

**대용량 미디어 파일을 자주 다루는 사람.** 동영상 컬렉터, 토렌트 사용자, 해외 스트리밍에 관심 있는 사람에게는 PikPak이 훨씬 현실적인 선택지다. 내 인터넷 대역폭을 쓰지 않고, 내 기기가 잠들어 있는 동안에도 파일을 받아두는 방식은 꽤 편리하다.

한국 커뮤니티에서도 "토렌트 자료 보관용으로 안성맞춤"이라는 평가가 많다. 10TB라는 용량 자체가 일반적인 개인 사용자 기준으로는 당분간 꽉 찰 일이 없다. HD 영상 기준으로 8,000개 이상을 저장할 수 있다.

Telegram 봇 연동 기능도 독특하다. @PikPak_Bot에 마그넷 링크를 던지면 알아서 클라우드에 저장된다. 텔레그램을 자주 쓰는 사람에게는 굉장히 자연스러운 흐름이다.

또한 PikPak은 WebDAV를 지원해서 rclone 같은 툴로 마운트하거나 NAS와 연동하는 것도 가능하다. 데이터를 직접 관리하고 싶은 파워유저들에게도 활용 여지가 있다.

### Dropbox가 맞는 사람

**문서 작업, 팀 협업, 로컬 파일 자동 백업이 필요한 사람.** Dropbox는 로컬 폴더를 클라우드와 실시간 동기화하는 것에 특화되어 있다. 회사에서 쓰는 파일, 공동 작업 중인 문서, 여러 기기에서 같은 파일을 열어야 하는 상황이라면 Dropbox가 훨씬 자연스럽다.

PDF 편집, 전자 서명, 파일 링크를 외부 사람에게 공유하는 기능도 Dropbox가 앞선다. 비즈니스 환경에서 파일 거버넌스나 팀 관리 기능이 필요하다면 Dropbox Standard가 현실적인 선택이다.

## PikPak의 단점도 알아두자

솔직하게 이야기하자면, PikPak에는 몇 가지 알아두면 좋은 점이 있다.

**로컬 동기화가 없다.** 내 컴퓨터 바탕화면 파일이 자동으로 PikPak에 올라가지 않는다. PikPak은 내가 직접 업로드하거나, 링크로 클라우드에 저장하는 방식이다. 사진이나 문서 자동 백업을 원한다면 Dropbox나 다른 서비스를 병행해야 한다.

**파일 공유가 제한적이다.** PikPak은 기본적으로 개인 전용 클라우드에 가깝다. 공유 링크를 보내도 상대방이 PikPak 계정을 만들어야 접근할 수 있는 경우가 있다. 외부 사람에게 파일을 편하게 공유하고 싶다면 Dropbox가 낫다.

**업로드 제한이 있다.** 웹 버전은 파일당 4GB, PC 앱은 파일당 20GB 제한이 있고, 하루 최대 1,000개 파일을 업로드할 수 있다. 일반적인 사용 범위에서는 크게 문제가 되지 않지만, 대규모 업로드가 필요한 상황에서는 인지하고 있어야 한다.

**신뢰도 이슈.** 2021년 출시된 비교적 새로운 서비스이고, 한국 커뮤니티에서 서버 화재 이후 일부 파일 접근 문제가 보고된 적도 있다. 정말 중요한 유일한 사본은 PikPak에만 두지 않는 것이 안전하다.

## 결론: PikPak과 Dropbox는 경쟁 관계가 아니다

두 서비스를 오래 들여다보면 결국 한 가지 결론에 도달한다. PikPak과 Dropbox는 사실 같은 카테고리에서 경쟁하는 서비스가 아니다.

Dropbox는 **파일 동기화와 협업 도구**다. 문서를 함께 만들고, 팀과 공유하고, 내 기기들을 한 상태로 유지하는 데 집중한다.

PikPak은 **클라우드 다운로더 겸 미디어 창고**다. 인터넷 어딘가에 있는 파일을 내 클라우드로 끌어당기고, 저장된 영상을 어디서든 스트리밍으로 보는 데 집중한다.

만약 두 가지 용도가 모두 필요하다면, 사실 두 서비스를 같이 쓰는 것이 가장 현실적인 답이다. PikPak 무료 플랜은 신용카드 없이 시작할 수 있으니, 일단 써보는 것 자체는 부담이 없다.
